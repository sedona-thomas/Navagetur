Password Safety

How does this widget work?

This widget takes in a password and returns an estimate of how long it would take to brute force crack this password on the current device.

Warning

If you are using this application on a website, do not enter your real passwords. Instead, go to the github repository and download the application to use on your personal device. When run locally, all processing of passwords is done on device and passwords are never stored.

Know Your Rights

More information: Electronic Frontier Foundation Know Your Rights

Why do secure passwords matter?

Your password is the biggest line of defense against your data being accessed by hackers and cybercriminals. The more secure your passwords are, the more secure your data is.
